HTML 기본구조

HTML5에서는 웹 페이지의 레이아웃만을 위한 별도의 새로운 Tag들을 제공합니다. 이 Tag들을 의미(Semantic) Tag라고 합니다.

의미(Semantic) Tag

의미(Semantic) Tag란 말 웹브라우저나 개발자에게 명확한 의미를 전달하는 Tag 입니다. 예를 들어 <header> Tag를 보면 의미적으로 헤더를 정의하고 웹브라우저, 개발자에게 헤더부라고 명시하는 역할을 합니다. 하지만 주의할 점은 <header> Tag를 쓴다고 해서 이미 그려진 레이아웃을 만들어주지 않습니다. 단지 의미를 부여할 뿐인 레이아웃은 CSS3로 작업합니다.

HTML5에서 추가된 의미(Semantic) Tag

  • <header>: HTML 헤더나 section에 헤더를 정의함.
  • <nav>: HTML 메뉴바를 정의함.
  • <main>: HTML 전체 Content를 정의함.
  • <section>: HTML Section을 정의함.
  • <article>: HTML 하나의 글(article)을 정의함.
  • <aside>: 웹 페이지에서 베너나 추가적인 문구 등을 정의함.
  • <footer>: HTML Footer를 정의함.
